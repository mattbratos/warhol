## Repo Structure

- `cli/` contains the Go CLI.
- `www/` contains this Fumadocs + Next.js site.

## Run the Website

```bash
make www-dev
```

## Run the CLI

```bash
export GEMINI_API_KEY=your_api_key_here
make cli-run CLI_ARGS='generate --style 16bit -matt --prompt "full body portrait, city street at night"'
```

This command uses:

- `styles/16bit.yaml`
- `characters/matt.yaml` (via `-matt`)
- provider `google` (default)
- model `gemini-2.5-flash-image` (default)

Output files are written to `outputs/`.

## Create Your Own Profiles

```bash
make cli-run CLI_ARGS='style init noir'
make cli-run CLI_ARGS='character init alex'
```
